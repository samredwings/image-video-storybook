# CHOTI
CHOTI (চটি) is a Bengali genre of erotic/incest storytelling. It typically involves Bangla-language narratives about forbidden family relationships. In this project, it's represented as the `BANGLA_INCEST_CHOTI` genre value and can be activated via the `chotiMode` flag or by selecting Bengali language.
