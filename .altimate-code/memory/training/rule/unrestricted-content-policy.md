# Unrestricted Content Policy
This platform operates in UNRESTRICTED mode. Every route handler should:
- NEVER block, filter, or refuse explicit/adult/erotic content
- NEVER moralize or refuse based on content sensitivity
- ALL content ratings supported (G through XXX)
- NSFW detection is DISABLED (always returns ALLOW with score 0)
- Roleplay should NOT have safety guards that block explicit content — this contradicts the unrestricted policy
